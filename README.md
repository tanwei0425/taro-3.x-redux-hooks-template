# Taro 3.x 项目模板

## taro-3.x-redux-hooks-template

一个可用于生产环境的基于 Taro3 的 React 框架多端项目模版。

## 技术栈

- Taro3x
- React
- Redux
- Taro-UI
- Axios
- Ahooks

## 功能列表

- 基础功能
  - [x] Sass 支持, 基础公用文件
  - [x] Less 支持, 基础公用文件
  - [x] 状态管理(redux)
  - [x] ahooks(阿里hooks库)
- 接口请求
  - [x] axios 封装
  - [x] 拦截器
- 工程化
  - [x] router 封装
